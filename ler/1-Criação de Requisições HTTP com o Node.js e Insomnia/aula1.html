Aula 1 - Instalação das Ferramentas - Visual Studio code, Node.js, Insomnia e Nodemon

Obs: O ambiente de desenvolvimento utilizado neste curso é o Windows 10. As ferramentas, comandos, e demais processos envolvidos referem-se a data de criação deste curso, em Setembro de 2024. Consulte a documentação oficial das ferramentas para verificar possíveis atualizações no decorrer dos anos, assim como o uso de diferente sistema operacional.

Visual Studio Code O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft. Ele é leve, rápido e altamente personalizável, tornando-se uma escolha popular entre os desenvolvedores. Com suporte para várias linguagens de programação e uma ampla gama de extensões, o Visual Studio Code oferece uma experiência de desenvolvimento poderosa e eficiente. Você pode fazer o download do Visual Studio Code no site oficial: https://code.visualstudio.com/
Node.js O Node.js é uma plataforma de desenvolvimento de software de código aberto que permite a execução de código JavaScript no servidor. Ele utiliza o motor V8 do Google Chrome para executar o código JavaScript de forma eficiente e escalável. Com o Node.js, os desenvolvedores podem criar aplicativos de rede rápidos e escaláveis, bem como APIs RESTful e serviços web. Você pode fazer o download do Node.js no site oficial: https://nodejs.org/
Insomnia O Insomnia é uma ferramenta de teste de API que permite aos desenvolvedores criar, executar e depurar solicitações HTTP de forma fácil e eficiente. Com o Insomnia, os desenvolvedores podem testar APIs RESTful, dentre outras, bem como monitorar e compartilhar solicitações e respostas de API. Você pode fazer o download do Insomnia no site oficial: https://insomnia.rest/
Utilize a versão gratuita do Insomnia. Será solicitado um cadastro na ferramenta para download e utilização. Sempre que for utilizar o Insomnia, faça o login com o e-mail e senha cadastrados.
Nodemon O Nodemon é uma ferramenta que ajuda os desenvolvedores a automatizar o processo de reinicialização de aplicativos Node.js durante o desenvolvimento. Com o Nodemon, os desenvolvedores podem fazer alterações no código-fonte do aplicativo e ver as alterações refletidas automaticamente, sem a necessidade de reiniciar o servidor manualmente. Você pode instalar o Nodemon globalmente usando o npm (Node Package Manager) com o seguinte comando: npm install -g nodemon
