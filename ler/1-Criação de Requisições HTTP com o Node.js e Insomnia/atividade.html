Atividade para Certificação

  1. Crie um Novo Projeto em Node.js com o nome de atividade_api_restfull de acordo com o conteúdo abordado no curso para a construção de Projeto Node
  2. Desenvolva uma API RestFull com os métodos HTTP GET, POST, PUT e DELETE com o tema: Produtos
  3. Utilize o Insomnia para testar as requisições HTTP
  4. Crie as requisições do tipo GET com
    1. Rota inicial
    2. Rota com programação utilizando o arrow function
    3. Rota com retorno em json
    4. Rota com Query Params
    5. Rota com Route Params
    6. Rota com Request Body
    7. Crie um array de dados de Produtos e crie as rotas abaixo vinculadas ao array
    8. Rota com Retorno de todos os produtos
    9. Rota com Retorno de um produto específico
    10. Rota com POST no array
    11. Rota com PUT no array
    12. Rota com DELETE no array
