GPO
Server Manager -> Ferramentas -> Gerenciamento de Política de Grupo
Floresta dominio.local

Domínios (Domains)

dominio.local
Já existe uma política padrão que é:
	Default Domain Policy (Política padrão já criada)

Poderá ser criada Políticas de Grupo dentro de cada Unidade Organizacional

Existe Política tanto para computadores quanto para usuários

Selecione a política padrão e com o botão direito do mouse e clique em editar, irá aparecer as políticas para computadores e políticas para usuários

GPO 1 - Criação de uma pasta compartilha
	1º Crie uma pasta no Windows Server em Disco Local c -> "Compartilhada-Rede"
	2º Compartilhe essa pasta na rede (Todos os usuários e Leitura/Escrita) e salve o caminho do compartilhamento da Pasta 
		-> \\SERVER-AD\Compartilhada-Rede

	Criação de Um política que será aplicada a todos as unidades organizacionais:
	1. Clique com o botão direito do mouse sobre o domínio
	2. Criar um GPO
	3. Insira um nome pertinente a função da política "Caminho de rede para pasta pública)
	4. Botão direito sobre a GPO e EDITAR
	5. Configurações de Usuários -> Preferência -> Configurações do Windows 
	-> Mapas de unidade (Drive Map) -> Botão direito do mouse sobre a área de mapas em branco -> Novo -> Unidade Mapeada -> Ação: Atualizar -> local: (Caminho da pasta na rede) \\SERVER-AD\Compartilhada-Rede -> Reconectar -> Usar "Escolha uma letra" -> aplicar
